Boleto Bancário - Parámetros de Pagamento

Este módulo permite-lhe receber pagamentos seguros por boleto bancário.
Se o cliente escolher pagar por boleto bancário, o estado da encomenda mudará para "Aguardando Pagamento"
Posto isto, terá de confirmar manualmente a encomenda após receber ou dar baixa no boleto.
